# BhoomiKonnect

BhoomiKonnect is a premium, SEO-first real estate marketplace prototype built with Next.js App Router, TypeScript, Tailwind CSS, shadcn-style UI primitives, Framer Motion, and Supabase-ready architecture.

## Scripts

```bash
pnpm install
pnpm dev
pnpm build
pnpm typecheck
```

## Environment

Create `.env.local` when connecting Supabase or Google Maps:

```bash
NEXT_PUBLIC_SITE_URL=https://bhoomikonnect.com
NEXT_PUBLIC_SUPABASE_URL=
NEXT_PUBLIC_SUPABASE_ANON_KEY=
NEXT_PUBLIC_GOOGLE_MAPS_API_KEY=
```
